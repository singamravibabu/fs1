# **The continue statement** in **for** and **while**
---

# **A. continue in for Loops (10 examples)**

---

## **1. Skip one value in a loop**

```python
for i in [1, 2, 3, 4, 5]:
    if i == 3:
        continue
    print(i)
```

Explanation: Skips 3 and prints others.

---

## **2. Skip vowels in a string**

```python
for ch in "education":
    if ch in "aeiou":
        continue
    print(ch)
```

Explanation: Prints only consonants.

---

## **3. Skip even numbers**

```python
for n in range(1, 10):
    if n % 2 == 0:
        continue
    print(n)
```

Explanation: Only odd numbers print.

---

## **4. Skip negative numbers**

```python
for n in [5, -2, 8, -1, 3]:
    if n < 0:
        continue
    print(n)
```

Explanation: Ignores negative values.

---

## **5. Skip short words**

```python
for word in ["hi", "python", "go", "learn"]:
    if len(word) < 3:
        continue
    print(word)
```

Explanation: Prints words with length ≥ 3.

---

## **6. Skip printing letter 'h'**

```python
for ch in "Python":
    if ch == "h":
        continue
    print(ch)
```

Explanation: Skips 'h'.

---

## **7. Skip items starting with letter ‘b’**

```python
for fruit in ["apple", "banana", "cherry", "berry"]:
    if fruit.startswith("b"):
        continue
    print(fruit)
```

Explanation: Prints items not starting with 'b'.

---

## **8. Skip duplicates**

```python
seen = set()
for n in [1, 2, 2, 3, 3]:
    if n in seen:
        continue
    seen.add(n)
    print(n)
```

Explanation: Prints only unique items.

---

## **9. Skip empty strings**

```python
for s in ["hello", "", "world", ""]:
    if s == "":
        continue
    print(s)
```

Explanation: Only non-empty strings print.

---

## **10. Skip values less than 10**

```python
for n in [5, 12, 3, 20, 1]:
    if n < 10:
        continue
    print(n)
```

Explanation: Prints only values ≥ 10.

---

# **B. continue in while Loops (10 examples)**

---

## **11. Skip number 3**

```python
i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
```

Explanation: Skips 3.

---

## **12. Skip even numbers**

```python
i = 0
while i < 10:
    i += 1
    if i % 2 == 0:
        continue
    print(i)
```

Explanation: Prints odd numbers only.

---

## **13. Skip values until a threshold**

```python
x = 0
while x < 10:
    x += 1
    if x < 5:
        continue
    print(x)
```

Explanation: Skips 1–4, prints 5–10.

---

## **14. Skip negative values**

```python
numbers = [3, -1, 4, -2, 5]
i = 0
while i < len(numbers):
    if numbers[i] < 0:
        i += 1
        continue
    print(numbers[i])
    i += 1
```

Explanation: Skips negative numbers.

---

## **15. Skip index 2**

```python
i = 0
while i < 5:
    if i == 2:
        i += 1
        continue
    print(i)
    i += 1
```

Explanation: Skips index 2.

---

## **16. Skip printing letter 'o'**

```python
text = "Python"
i = 0
while i < len(text):
    if text[i] == "o":
        i += 1
        continue
    print(text[i])
    i += 1
```

Explanation: Skips the letter 'o'.

---

## **17. Skip when value is zero**

```python
values = [0, 5, 0, 10]
i = 0
while i < len(values):
    if values[i] == 0:
        i += 1
        continue
    print(values[i])
    i += 1
```

Explanation: Skips zeros in the list.

---

## **18. Skip when input is empty**

```python
text = ""
while text != "stop":
    text = input("Enter text: ")
    if text == "":
        continue
    print("You typed:", text)
```

Explanation: Empty input is ignored.

---

## **19. Skip numbers divisible by 3**

```python
i = 1
while i <= 10:
    if i % 3 == 0:
        i += 1
        continue
    print(i)
    i += 1
```

Explanation: Skips 3, 6, 9.

---

## **20. Skip values after increment**

```python
i = 0
while i < 7:
    i += 1
    if i == 5:
        continue
    print("Value:", i)
```

Explanation: Skips 5.
