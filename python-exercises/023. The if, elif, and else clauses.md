# **The full if statement**

---

## 1. Basic if–elif–else

```python
if 5 > 10:
    print("Five is bigger")
elif 10 > 5:
    print("Ten is bigger")
else:
    print("Numbers are equal")
```

Explanation: First false, second true, so second block runs.

---

## 2. Checking grades

```python
score = 85
if score >= 90:
    print("A grade")
elif score >= 80:
    print("B grade")
else:
    print("C or below")
```

Explanation: 85 matches the second condition.
